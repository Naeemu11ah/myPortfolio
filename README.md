# My Portfolio

This responsive portfolio made for mobile and laptop screens using React Vite and tailwind CSS

**What it haves**

- My Introduction
- My Education
- My Technical Skills
- The Projects I made
- My contacts & Social media Links

**How to use**

- Clone the Portfolio
- npm install
- npm run dev
